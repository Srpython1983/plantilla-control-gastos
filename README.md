# 📊 Plantilla de Control de Gastos — Gratis

Herramienta práctica para registrar, categorizar y controlar 
todos los gastos de tu empresa mes a mes. Diseñada para dueños 
de negocio, emprendedores y equipos administrativos que quieren 
saber exactamente en qué se va el dinero.

---

## ✅ ¿Qué incluye esta versión gratuita?

- Registro de gastos diarios por categoría
- Control de gastos fijos vs variables
- Límite de gasto mensual por categoría
- Alerta automática cuando superas el límite
- Resumen mensual automático
- Fácil de usar — sin conocimientos técnicos

## 🚀 ¿Cómo usarla?

1. Descarga el archivo `control-gastos-basico.xlsx`
2. Abre con Excel o Google Sheets
3. Define tu presupuesto mensual por categoría
4. Registra cada gasto con fecha, categoría y monto
5. El resumen y las alertas se actualizan automáticamente
